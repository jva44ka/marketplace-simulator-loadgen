# marketplace-simulator-loadgen

Генератор нагрузки — часть учебного проекта «Симулятор маркетплейса».

Симулирует реальный трафик: пополнение товаров через Kafka-события, оформление заказов (order flow) и просмотр корзин. Контроль RPS через token bucket.

→ [Подробная документация](docs/README.md)

Запуск в составе полного стека: [marketplace-simulator](https://github.com/jva44ka/marketplace-simulator)
